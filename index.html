What is Project Rooted?

Project rooted is where custom roms are taken, extracted, and repackaged with patched images that allow root access. This means that when you download a rom from us it will be automatically rooted.

About

I, Jacob, create these pre-patched rom files whenever I hop to a different rom or updating whatever I'm currently running. Wanted to share these files with the community that may also want these files.
